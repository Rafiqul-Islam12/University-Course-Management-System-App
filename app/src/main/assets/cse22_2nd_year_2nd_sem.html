CSE 2201: Database Management Systems

Credit: 3.00

Course Objectives:

To teach the fundamentals of data structure techniques and their applications.

Course Outline:

Concept of Data Structure, ADT; Linear & Non-Linear Data Structures; Graph, Sorting, Hashing.

Recommended Books:

1. Lipschutz – Data Structures (Schaum’s)
2. Horowitz et al. – Fundamentals of DS in C
3. Kruse & Leung – DS and Program Design in C

CSE 2202: Database Management Systems LAB

Credit: 1.50

Course Objectives:

To implement various data structures and algorithms in C.

Course Outline:

Implementing Arrays, Stack, Queue, Linked List, Tree, Graph, Sorting, Searching.

Recommended Books:

Same as CSE 2201.

CSE 2203: Computer Architecture and Organization

Credit: 3.00

Course Objectives:

To understand CPU architecture, memory, subsystems and instruction lifecycle.

Course Outline:

Processor Design, ALU, Control Unit, System Org, Parallel Processing, RISC.

Recommended Books:

1. Hamacher, Zaky
2. J.P. Hayes
3. W. Stallings

CSE 2204: Design Pattern

Credit: 3.00

Course Objectives:

Learn software design concepts and pattern-based solutions.

Course Outline:

Patterns, MVC, Case Studies, Creational, Structural, Behavioral Patterns.

Recommended Books:

1. Erich Gamma – Design Patterns
2. Mark Grand – Patterns in Java
3. Mark Grand – Java Enterprise Design Patterns
